# Go Google Auth Frontend

Vue.js 3 + TypeScript を使用したフロントエンドアプリケーション

## 機能

- Cookie/Session テスト機能
- バックエンドとの通信テスト
- レスポンシブデザイン

## セットアップ

```bash
# 依存パッケージのインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build

# プレビュー
npm run preview
```

## 開発環境

- Node.js >= 22.15.0
- npm

## 技術スタック

- Vue 3
- TypeScript
- Vite
- Vue Router
